redup_servicekit package

Toolkit for Python microservices: config, logging, Prometheus metrics, gRPC.

The redup_servicekit package contains:

  • redup_servicekit.config — YAML config singleton

  • redup_servicekit.logging — JSON console logging

  • redup_servicekit.health — gRPC health checks

  • redup_servicekit.metrics — Prometheus registry and constants

  • redup_servicekit.monitoring — MonitorServer, MetricServer, task stats

  • redup_servicekit.grpc — async client and server decorators
